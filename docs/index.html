Open Source AI Media Studio

Local AI Media Platform

A Dockerized FastAPI, nginx, and Stable Diffusion stack for prompt-to-video generation, local narration fallback, and multi-track MP4 timeline export.

View Repository

Project Preview

This static page gives visitors the project overview. The full generation and editor experience runs locally through Docker.

What Is Included

  • FastAPI backend with job queue and editor APIs
  • Static frontend served by nginx with /api proxying
  • Stable Diffusion worker with a fast smoke-test mode

Verified Workflow

The Docker stack builds, health checks pass, prompt-to-video jobs complete, and captioned editor exports download as MP4.

Feature Snapshot

Prompt to Video Jobs Replicate and ElevenLabs Integrations Local Fallback Pipeline GPU Auto Detection (CUDA/MPS/CPU) Multi-Track Timeline Trim, Split, Transitions Audio Waveform Timeline Caption Tracks MP4 Export

Run Locally

git clone https://github.com/hridayeshp/local-ai-media-platform.git
cd local-ai-media-platform
docker compose up --build -d
# open http://localhost

Use SD_MOCK=true docker compose up --build -d for a quick full-stack smoke test without downloading Stable Diffusion weights.
